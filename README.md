# node-es6-tdd
node.js + es6 project scaffold for tdd

## install
```
git clone https://github.com/raydecastro/node-es6-tdd.git
cd node-es6-tdd
npm install
```

## test
```
npm test
gulp test
gulp test-auto
```

## technologies
* javascript
* es6 + babel
* mocha + chai
* node + npm 
* gulp
